# Leather Craft Ventures Ltd - Website

Professional website for Leather Craft Ventures Ltd, Nigeria's specialists in premium leather goods and corporate apparel manufacturing.

## 🏢 About

Leather Craft Ventures Ltd specializes in:
- **Security Apparells** - Professional leather boots and gear
- **Uniforms** - School and corporate uniform manufacturing  
- **Knitted Collars** - Precision-crafted collar accessories
- **Monogramming & Customisation** - Brand identity services

## 🚀 Features

- **Responsive Design** - Mobile-first approach
- **Accessibility** - WCAG 2.1 AA compliant
- **SEO Optimized** - Meta tags, structured data
- **Dark Mode** - Auto-detection support
- **Performance** - Optimized loading and fonts
- **Security** - CSP headers and best practices

## 🎨 Design System

- **Fonts**: Montserrat (headings) + Open Sans (body)
- **Colors**: Navy (#1B4882), Crimson (#CE2939), Gold (#C8A05A)
- **Typography**: Professional hierarchy with proper contrast

## 📱 Pages

- **Home** - Hero section with company overview
- **About** - Company story and mission
- **Services** - Detailed service offerings
- **Portfolio** - Client work showcase
- **Contact** - Quote request form

## 🛠 Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript ES6
- **Fonts**: Google Fonts
- **Deployment**: GitHub Pages
- **Browser Support**: Modern browsers
